# AI-Based Early Disease Detection System

A smart health monitoring web application designed for early risk awareness using user-input vitals, lifestyle data, and symptoms.

## Features
- Patient profile management
- Live vitals input with animated heartbeat
- 3D human body visualization
- Health data & symptoms tracking
- AI-generated health summary
- Futuristic, responsive UI

## Tech Stack
- HTML
- CSS (Glassmorphism + Animations)
- JavaScript
- Three.js

## Disclaimer
This project is a prototype built for academic and hackathon purposes only.
It does NOT provide medical diagnosis. Always consult a medical professional.
