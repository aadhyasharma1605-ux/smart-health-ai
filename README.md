
AI-Based Early Disease Detection System

Smart Health+ is a web-based AI-inspired health screening platform designed to assist in early detection of potential health risks using patient vitals, lifestyle data, symptoms, and clinical indicators.
This project was developed as part of a hackathon submission and focuses on usability, visualization, and preventive healthcare insights.
